# STM32 智能电压监控与报警系统

基于正点原子精英版（STM32F103ZET6）开发的电压监控装置，具备模拟电压采集、动态阈值调节、声光报警、OLED实时显示及串口上报功能。

## 功能特性

-  ADC 采集电位器电压（PA0），**10次平均滤波**，波动从 ±50mV 降至 ±5mV
-  **按键**（KEY0/KEY1）动态调节报警阈值，步长 100
-  超限时 **LED 点亮 + 有源蜂鸣器鸣响**，响应时间 <5ms
-  **0.96 寸 OLED（I2C 接口）** 实时显示电压、阈值、报警状态
-  **USART1** 每 200ms 向上位机打印数据（ADC 值、电压、阈值、报警状态）

## 硬件连接

| 外设        | 引脚         |
|-------------|--------------|
| ADC 输入     | PA0          |
| LED         | PA8          |
| 蜂鸣器      | PB8          |
| KEY0        | PE3          |
| KEY1        | PE4          |
| OLED SCL    | PB6          |
| OLED SDA    | PB7          |
| USART1 TX   | PA9          |
| USART1 RX   | PA10         |

## 如何运行

1.  用 Keil MDK 打开工程，编译并烧录到开发板
2.  打开串口助手，波特率设置为 **115200**（若乱码可尝试 9600 或 4800）
3.  旋转电位器观察 OLED 上的电压变化，按 KEY0/KEY1 调整阈值触发报警

## 代码结构

- `main.c` - 主程序逻辑
- `adc.c` / `adc.h` - ADC 驱动（10次平均滤波）
- `led.c` / `led.h` - LED 驱动
- `beep.c` / `beep.h` - 蜂鸣器驱动
- `oled.c` / `oled.h` / `oled_font.h` - I2C OLED 驱动
- `usart.c` / `usart.h` - 串口通信与 printf 重定向

## 开发环境

-  IDE: Keil MDK 5
-  库版本: HAL 库
-  编译器: ARMCC
